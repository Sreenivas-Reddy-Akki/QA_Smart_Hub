Functions (Intermediate Concepts)

In Python, functions can accept variable numbers of arguments, return multiple values, and even reference themselves recursively. You can also create anonymous functions (lambdas).

1️⃣ *args and **kwargs

- *args allows you to pass multiple positional arguments. - **kwargs allows you to pass multiple keyword arguments. 


# Example: Using *args
def add_all(*numbers):
    total = 0
    for n in numbers:
        total += n
    return total

print(add_all(5, 10, 15, 20))

# Example: Using **kwargs
def show_details(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

show_details(name="Alice", age=25, city="New York")

2️⃣ Recursion

Recursion occurs when a function calls itself to solve smaller instances of a problem. 


# Example: Factorial using recursion
def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print("Factorial of 5 is:", factorial(5))

3️⃣ Lambda Functions

Lambda functions are small, anonymous functions defined using the lambda keyword. 


# Example: Lambda function
square = lambda x: x ** 2
print("Square of 6:", square(6))

# Example: Lambda with map() and filter()
nums = [1, 2, 3, 4, 5, 6]
even_nums = list(filter(lambda x: x % 2 == 0, nums))
squares = list(map(lambda x: x ** 2, nums))

print("Even numbers:", even_nums)
print("Squares:", squares)

File Handling in Python

Python allows reading and writing files using built-in functions. You can open a file using the open() function, perform operations, and then close it.

1️⃣ Reading Files


# Example: Reading a file
file = open("sample.txt", "r")
content = file.read()
print(content)
file.close()

2️⃣ Writing to Files


# Example: Writing to a file
file = open("output.txt", "w")
file.write("Hello, Python File Handling!")
file.close()

3️⃣ Appending Data


# Example: Appending data
file = open("output.txt", "a")
file.write("\nThis line was appended later.")
file.close()

4️⃣ Using with open()

It’s best practice to use with open() because it automatically closes the file. 


# Example: File handling with 'with' statement
with open("data.txt", "r") as file:
    for line in file:
        print(line.strip())

You can also handle errors using try-except while dealing with files to prevent crashes.

Modules & Packages

A module is a Python file containing functions, variables, or classes. A package is a collection of modules organized inside a folder with an __init__.py file.

1️⃣ Importing a Module


# math_module.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b
  

# main.py
import math_module

print("Sum:", math_module.add(5, 3))
print("Difference:", math_module.subtract(10, 4))

2️⃣ Built-in Modules

Python includes many useful built-in modules such as math, random, datetime, and os.


import math, random, datetime, os

print("Square root of 16:", math.sqrt(16))
print("Random number:", random.randint(1, 100))
print("Today:", datetime.date.today())
print("Current working directory:", os.getcwd())

3️⃣ Creating Packages

A package is just a folder with an __init__.py file.


# project/
# ├── mypackage/
# │   ├── __init__.py
# │   ├── operations.py
# └── main.py

# mypackage/operations.py
def multiply(a, b):
    return a * b

# main.py
from mypackage import operations
print("Product:", operations.multiply(3, 4))

Object-Oriented Programming (OOP)

OOP allows you to structure programs using classes and objects. It supports key concepts like Encapsulation, Inheritance, and Polymorphism.

1️⃣ Classes and Objects


# Example: Creating a class and object
class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model

    def display_info(self):
        print(f"Car: {self.brand} {self.model}")

# Object creation
car1 = Car("Tesla", "Model S")
car1.display_info()

2️⃣ Instance and Class Variables


class Student:
    school_name = "Python Academy"  # Class variable

    def __init__(self, name, age):
        self.name = name
        self.age = age  # Instance variable

    def show(self):
        print(f"{self.name}, {self.age}, School: {Student.school_name}")

s1 = Student("Alice", 20)
s1.show()

3️⃣ Inheritance

Inheritance lets one class (child) use attributes and methods of another class (parent). 


# Example: Single inheritance
class Animal:
    def speak(self):
        print("Animal speaking...")

class Dog(Animal):
    def bark(self):
        print("Dog barking...")

dog1 = Dog()
dog1.speak()
dog1.bark()

4️⃣ Method Overriding

A subclass can override a method from its parent to change its behavior.


class Animal:
    def sound(self):
        print("Some sound")

class Cat(Animal):
    def sound(self):
        print("Meow!")

c = Cat()
c.sound()

5️⃣ Polymorphism

Polymorphism allows the same method to behave differently for different classes.


class Bird:
    def move(self):
        print("Flying in the sky")

class Fish:
    def move(self):
        print("Swimming in the water")

for animal in (Bird(), Fish()):
    animal.move()

6️⃣ Encapsulation

Encapsulation restricts direct access to variables and methods to protect data integrity.


class Account:
    def __init__(self, owner, balance):
        self.owner = owner
        self.__balance = balance  # private variable

    def deposit(self, amount):
        self.__balance += amount
        print("Updated balance:", self.__balance)

acc = Account("John", 1000)
acc.deposit(500)
# print(acc.__balance)  # ❌ will raise error (private)

Object-Oriented Programming (OOP)

Python supports object-oriented programming, which allows you to model real-world entities using classes and objects.

Creating a Class and Object

class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model

    def display(self):

Inheritance

Inheritance allows one class (child) to acquire the properties and behaviors of another class (parent).

Single Inheritance Example

class Vehicle:
    def info(self):
        print("This is a vehicle.")

class Car(Vehicle):
    def brand(self):
        print("Brand: Tesla")

my_car = Car()
my_car.info()
my_car.brand()

Python also supports multiple inheritance, where a class can inherit from more than one parent class.

Example of Polymorphism

class Dog:
    def sound(self):
        print("Bark")

class Cat:
    def sound(self):
        print("Meow")

for animal in [Dog(), Cat()]:
    animal.sound()

Iterators in Python

Iterators are objects that allow you to traverse through all elements of a collection (like lists or tuples). Any object with __iter__() and __next__() methods is considered an iterator.

Example: Using an Iterator

numbers = [1, 2, 3]
it = iter(numbers)

print(next(it))  # 1
print(next(it))  # 2
print(next(it))  # 3

Once all elements are traversed, calling next() again raises a StopIteration exception.

Generators

Generators are a simple way to create iterators using functions and the yield keyword. They allow you to generate values on the fly without storing the entire sequence in memory.

Example: Generator Function

def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

for num in count_up_to(5):
    print(num)

Generators are memory-efficient and great for working with large data or infinite sequences.

Decorators

Decorators are functions that modify the behavior of another function without permanently changing it. They are often used for logging, access control, or performance measurement.

Example: Function Decorator

def decorator_function(func):
    def wrapper():
        print("Before the function runs...")
        func()
        print("After the function runs...")
    return wrapper

@decorator_function
def say_hello():
    print("Hello, Python!")

say_hello()

The @decorator_name syntax is shorthand for wrapping a function inside another.

Regular Expressions (Regex)

Regular expressions allow you to search, match, and manipulate strings using specific patterns. Python provides the re module for regex operations.

Example: Basic Regex Operations

import re

text = "Contact me at test@example.com or info@qa-smart.com"
emails = re.findall(r"[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}", text)
print(emails)

re.findall() returns all matching patterns as a list. Other useful functions include re.search(), re.match(), and re.sub().

Custom Exceptions

Python allows you to create your own exception types by defining custom classes that inherit from Exception.

Example: Creating a Custom Exception

class InvalidAgeError(Exception):
    pass

def check_age(age):
    if age < 18:
        raise InvalidAgeError("Age must be at least 18!")
    else:
        print("Access granted!")

try:
    check_age(16)
except InvalidAgeError as e:
    print("Error:", e)

Custom exceptions make your code more readable and help in better debugging of domain-specific errors.

Introduction to Python

  • Easy to learn and write.
  • Interpreted ‚Äî no need to compile.
  • Supports multiple paradigms (OOP, functional, procedural).

You can install Python from the official website python.org. After installation, verify it using your terminal or command prompt:

You can also use an IDE like PyCharm, VS Code, or Jupyter Notebook for development.

Python Syntax

Python uses indentation (spaces or tabs) to define code blocks instead of braces {}. The indentation level is critical ‚Äî incorrect spacing causes errors. 


# Example of correct indentation
if 5 > 2:
    print("Five is greater than two!")

# Incorrect indentation will raise an error
# if 5 > 2:
# print("Error!")  # ‚ùå Missing indentation

‚úÖ Always use 4 spaces for indentation (recommended).
‚úÖ Python statements end with a newline ‚Äî no semicolons needed (optional though).

Variables in Python

Variables store data in memory. You don‚Äôt need to declare their type ‚Äî Python infers it automatically.

‚úÖ Variable names must start with a letter or underscore.

Python Data Types

Python supports several built-in data types:

  • int ‚Äî integers (whole numbers)

  • str ‚Äî string of text

Control Flow (if, elif, else)

Control flow allows your program to make decisions. Python‚Äôs if, elif, and else statements decide which code block should execute. 


# Example: Grade Evaluation
marks = 85

if marks >= 90:
    print("Grade: A")
elif marks >= 75:
    print("Grade: B")
elif marks >= 60:
    print("Grade: C")
else:
    print("Grade: D")

‚úÖ The if statement checks the first condition.
‚úÖ elif checks additional conditions if the previous ones are false.
‚úÖ else executes only when none of the above are true.

Lists in Python

Lists store multiple items in a single variable. Lists are mutable (can be changed). 


# Example: Working with lists
fruits = ["apple", "banana", "cherry"]
print(fruits[0])          # Access first element
fruits.append("orange")   # Add element
fruits.remove("banana")   # Remove element

for fruit in fruits:
    print(fruit)

‚úÖ Lists use square brackets [].
‚úÖ Common methods: append(), insert(), remove(), sort(), reverse().

Tuples in Python

Tuples are ordered, immutable collections. Once defined, their values cannot be modified. 


# Example: Tuple operations

print(coordinates[0])
print(len(coordinates))
# coordinates[1] = 40  ‚ùå This will cause an error (tuples are immutable)

‚úÖ Tuples use parentheses ().
‚úÖ Tuples are faster than lists and great for fixed data.

Dictionaries in Python

Dictionaries store data as key-value pairs. Each key must be unique. 


# Example: Dictionary operations
student = {
    "name": "John",
    "age": 21,
    "course": "Python"
}

print(student["name"])
student["age"] = 22
student["city"] = "Hyderabad"

for key, value in student.items():
    print(key, ":", value)

‚úÖ Use {} to define dictionaries.
‚úÖ Access values with dict["key"].
‚úÖ Useful methods: keys(), values(), items().

Strings in Python

Strings represent text enclosed in single or double quotes. They are immutable. 


# Example: String operations

print(text.lower())
print(text.upper())
print(text.replace("Python", "Java"))

‚úÖ Strings are iterable (you can loop through them).
‚úÖ Use slicing (text[start:end]) to extract parts.
‚úÖ Common methods: lower(), upper(), replace(), split(), strip().

Sets in Python

Sets are unordered collections of unique items. They are useful for removing duplicates. 


# Example: Working with sets
numbers = {1, 2, 3, 4, 4, 5}
print(numbers)  # Duplicate 4 is automatically removed

numbers.add(6)
numbers.remove(3)
print("Updated set:", numbers)

setA = {1, 2, 3}
setB = {3, 4, 5}
print("Union:", setA | setB)
print("Intersection:", setA & setB)
print("Difference:", setA - setB)

‚úÖ Sets use curly braces {}.
‚úÖ Unordered ‚Äî items have no index.
‚úÖ Support mathematical operations like union, intersection, and difference.

Loops in Python

Loops are used to execute a block of code repeatedly until a specific condition is met. Python mainly supports two types of loops: for loop and while loop.

For Loop

The for loop is used to iterate over a sequence (like a list, tuple, dictionary, or string).


# Example: Using for loop
for i in range(5):
    print("Iteration:", i)

While Loop

The while loop continues running as long as the condition remains True.


# Example: Using while loop
count = 0
while count < 5:
    print("Count is:", count)
    count += 1

Break and Continue

break stops the loop immediately, while continue skips the current iteration and moves to the next. 


# Example of break and continue
for i in range(10):
    if i == 5:
        break  # stop the loop when i = 5
    if i % 2 == 0:
        continue  # skip even numbers
    print("Odd number:", i)

Functions in Python

Functions are reusable blocks of code designed to perform a specific task. You define them with the def keyword, and they can take arguments and return values.

Defining and Calling a Function


# Example: Basic function
def greet():
    print("Hello, welcome to Python learning!")

greet()

Function with Parameters


# Example: Function with parameters
def add_numbers(a, b):
    return a + b

result = add_numbers(10, 5)
print("Sum is:", result)

Default and Keyword Arguments


# Example: Default argument
def greet_user(name="Guest"):
    print("Hello,", name)

greet_user("Alice")  # custom name

# Example: Keyword arguments
def user_info(name, age):
    print(f"Name: {name}, Age: {age}")

user_info(age=25, name="John")

Return Values

A function can return data to the caller using the return statement. 


# Example: Returning values
def multiply(a, b):
    return a * b

product = multiply(4, 6)
print("Product:", product)

Python Data Structures

Data structures are used to store and organize data efficiently. The main built-in data structures in Python are Lists, Tuples, Sets, and Dictionaries.

Lists

Lists are ordered, mutable collections. They can store different data types.


# Example: Working with lists
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")

print("First fruit:", fruits[0])

Tuples

Tuples are ordered and immutable ‚Äî once created, their values cannot change.


# Example: Tuple
coordinates = (10, 20)
print("X:", coordinates[0])
print("Y:", coordinates[1])

Sets

Sets are unordered collections of unique items.


# Example: Set
colors = {"red", "blue", "green", "red"}
print(colors)  # 'red' appears only once
colors.add("yellow")
print(colors)

Dictionaries

Dictionaries store data in key-value pairs. They are mutable and unordered (in Python < 3.7). 


# Example: Dictionary
person = {"name": "Alice", "age": 25, "city": "New York"}
print("Name:", person["name"])
person["age"] = 26
print(person)

Strings in Python

Strings are sequences of characters enclosed in quotes. Python allows both single (' ') and double (" ") quotes to define strings.

String Creation and Access


# Creating strings
name = "Python Programming"
print(name)

# Accessing characters
print("First character:", name[0])

String Slicing

Slicing lets you extract parts of a string using the format string[start:end]. 


# Example: String slicing
text = "Learning Python"
print(text[0:8])   # 'Learning'
print(text[9:])    # 'Python'

String Methods

Python provides many built-in methods to work with strings.


# Example: String methods
message = " hello world "
print(message.upper())      # HELLO WORLD
print(message.lower())      # hello world
print(message.strip())      # removes extra spaces
print(message.replace("world", "Python"))  # hello Python
print("Does message start with 'h'? ->", message.startswith("h"))

String Formatting

String formatting helps to create dynamic messages. Python supports f-strings, format(), and concatenation. 


name = "Alice"
age = 25

# Using f-string
print(f"My name is {name} and I am {age} years old.")

# Using format()
print("My name is {} and I am {} years old.".format(name, age))

User Input and Output

Python uses the input() function to take user input and print() to display output. The input is always taken as a string unless explicitly converted. 


# Example: Taking user input
name = input("Enter your name: ")
print("Welcome,", name)

# Taking numeric input
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
print("Sum:", num1 + num2)

You can also customize the output with multiple values or separators. 


# Example: Customized print
print("Python", "is", "fun", sep=" - ", end="!\n")

Basic Exception Handling

Errors can occur at runtime, and Python provides try-except blocks to handle them gracefully, preventing the program from crashing.

Try-Except Block


# Example: Basic exception handling
try:
    number = int(input("Enter a number: "))
    print("You entered:", number)
except ValueError:
    print("That's not a valid number!")

Try-Except-Else-Finally

You can add optional else (runs if no error occurs) and finally (always runs) blocks. 


# Example: Full exception handling
try:
    num = int(input("Enter a number: "))
except ValueError:
    print("Invalid input! Please enter a number.")
else:
    print("Valid input! Number is:", num)
finally:
    print("Execution completed.")
